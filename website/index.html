🔧 Installation Guide

1

Install Python 3.10+

Download and install Python from python.org. Make sure to check "Add Python to PATH" during installation.

python --version
2

Download BloxBot

Download the source code or clone the repository.

git clone https://github.com/your-repo/bloxbot.git
cd bloxbot
3

Install Dependencies

Install all required packages with one command.

pip install -r requirements.txt
4

Launch BloxBot

Run the application and start creating!

python main.py
5

Build Executable (Optional)

Create a standalone .exe file that doesn't require Python.

python build.py
