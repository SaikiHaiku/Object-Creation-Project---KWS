🔧 Guide d'Installation

1

Installer Python 3.10+

Telechargez et installez Python depuis python.org. Cochez "Add Python to PATH" pendant l'installation.

python --version
2

Telecharger OCP

Telechargez le code source ou clonez le depot.

git clone https://github.com/SaikiHaiku/Object-Creation-Project---KWS.git
cd Object-Creation-Project---KWS
3

Installer les Dependances

Installez tous les packages requis en une commande.

pip install -r requirements.txt
4

Lancer OCP

Lancez l'application et commencez a creer !

python main.py
5

Build Executable (Optionnel)

Creez un fichier .exe autonome qui ne necessite pas Python.

python build.py
